Movie enthusiasts want to be able to access information about different movies, directors, and genres. The server-side of the myFlix application will ensure they have access to this information, that their requests can be processed, and that all necessary data can be stored.

Business Logic URL HTTP Method Query Parameters Request Body Data Format Response Body Data Format
Returns a list of all movies /movies GET None None A JSON object of all the movies
Returns data about single movie by title /movies/:movieTitle GET :title None A JSON object of a single movie
Returns data about a genre by name/title /movies/genres/:genreName GET :genreName None A JSON object of a specific genre
Returns data about a director by name /movies/directors/:directorName GET :directorName None A JSON object of a specific director movies
Register new user /users POST None A JSON object of a new user example { "Username": "Tom1", "Password": "Tomspassword", "Email": "tom@gmail.com", "Birthday": "07-03-1983" } A JSON object of the new user along with the an ID number and empty array for favorite movies example{ "Username": "Tom1", "Password": "Tomspassword", "Email": "tom@gmail.com", "Birthday": "07-03-1983T05:00:00.000Z" "FavoriteMovies": [], "_id": "64bffcee7b93c84cc18bb0db" }
Allows users to update their user info /users/:Username PUT None A JSON of the four user categories example { "Username": "Tom1", "Password": "Tomspassword", "Email": "tom@gmail.com", "Birthday": "07-03-1983"} A JSON with the everything that was changed, and what wasn't example{ "Username": "Tom1", "Password": "Tomspassword", "Email": "tom@gmail.com", "Birthday": "07-03-1983" "Birthday": "07-03-1983T05:00:00.000Z" "FavoriteMovies": [], "_id": "64bffcee7b93c84cc18bb0db"}
Allows a user to be deleted /users/:Username DELETE :Username None A message confirming the user has been deleted
Allows users to add a movie to their favorites /users/:Username/:movieTitle POST :Username/:movieTitle None A JSON returning with the the users id, username, password, email, birthday and the favorite movie being added to the array of other favorite movies
Allows users to remove a movie from their favorites /users/:Username/movies/:movieTitle DELETE :Username/:movieTitle None A message confirming the movie has been removed from the user's list
